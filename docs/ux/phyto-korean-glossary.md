# PhytoSync Korean Glossary

이 문서는 PhytoSync 프론트엔드에서 사용자에게 직접 보이는 용어의 기본 표기 기준이다.
화면, 카드, 버튼, 툴팁, 테이블 헤더, 차트 범례는 가능하면 이 표기를 따른다.

## 환경·제어

| 내부/기술 용어 | 화면 기본 표기 | 보조 표기 | 사용 원칙 |
|---|---|---|---|
| Temperature | 온도 | 하우스 온도 | 단순 온도보다 하우스 맥락이 중요할 때만 보조어 사용 |
| Relative Humidity | 습도 | 상대습도 | 메인 화면은 `습도`, 상세 화면만 `상대습도` 허용 |
| CO2 / CO₂ ppm | 이산화탄소 | CO₂ | 약어 단독 노출 금지 |
| PAR / PPFD | 광량 | PAR | 메인 KPI에서는 `광량` |
| Radiation / DLI | 누적광량 | 오늘 받은 빛 | 설명형 카드에서 쉬운 말 병기 |
| VPD | 공기 건조도 | 포차(VPD) | `VPD` 단독 표기 금지 |
| RTR | 빛 맞춤 온도 | RTR | 상세 비교 화면에서만 보조 표기 |
| Setpoint | 목표값 | 설정값 | 설정 화면 외에는 `목표값` 우선 |
| Ventilation | 환기 | 환기 열기 | 행동 문장에서는 동사형 허용 |
| Heating | 난방 | 난방 가동 | |
| Cooling | 냉방 | 냉방 가동 | |
| Screen | 커튼 | 보온·차광 커튼 | `스크린` 단독 지양 |
| Circulation fan | 유동팬 | 순환팬 | 현장 표기 우선 |
| Condensation risk | 결로 주의 | 이슬맺힘 주의 | |
